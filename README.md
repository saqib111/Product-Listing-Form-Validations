# React Mini Dashboard & Form Validation

This project is built as part of a Frontend Developer (React.js) assignment.  
It demonstrates clean UI implementation, state management, and form validation using modern React practices.

---

## 🚀 Features

### Task 1 – Product Listing (Mini Dashboard)

- Fetch data from API
- Display products in responsive grid layout
- Search (debounced)
- Sorting (name & price)
- Pagination
- Loading skeletons
- Error handling
- Empty state UI
- Smooth UI transitions

### Task 2 – Form + Validation

- Built with React Hook Form + Zod
- Full Name, Email, Phone Number, Password
- Inline validation errors
- International phone number input with country selector
- Proper formatting and validation
- Password visibility toggle
- Success state after submission

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- React Hook Form
- Zod
- Axios
- React Router DOM
